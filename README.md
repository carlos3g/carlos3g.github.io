### 🚀 How to run

1. Clone the repository

   ```bash
   git clone https://github.com/carlos3g/carlos3g.github.io.git
   ```

2. Install the dependencies

   ```bash
   cd carlos3g.github.io
   yarn install
   ```

3. Run the application
   ```bash
   yarn start
   ```
